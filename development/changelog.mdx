## Release Highlights

<Accordion title="Core Features">

### Knowledge Graph System
- Local-first knowledge graph built from emails, meetings, and notes
- Obsidian-compatible Markdown vault with backlinks
- Transparent, inspectable, and editable memory system
- Context accumulation over time with explicit relationships

### Integrations
- Gmail integration for email context
- Granola meeting notes integration
- Fireflies meeting notes integration
- Google Calendar and Drive support

### AI Capabilities
- Meeting prep from prior decisions and context
- Email drafting grounded in history
- Document and PDF slide generation
- Background agents for automated workflows
- Bring-your-own-model support (local and hosted)

### MCP Tool Support
- Model Context Protocol (MCP) integration
- Extensible tool system for external services
- Support for search, databases, CRMs, and custom tools

</Accordion>

<Accordion title="Platform Support">

### Supported Platforms
- macOS (Apple Silicon and Intel)
- Windows
- Linux

### Download Options
All platform builds are available on the [releases page](https://github.com/rowboatlabs/rowboat/releases/latest).

</Accordion>
